Rust Language Fundamentals

Why Rust?

### A low level language - Compiled, not interpreted - Enforces very strict rules - Goal: all errors at compile time (no segmentation faults) - Semi-optimal garbage collection
### Modern & high level features - Great compiler & error messages - Iterators - Functional programming features - Function pointers & closures - Higher order functions - Package manager - Testing framework
### Variable Consumption - One of Rust's most distinctive features - Allows the compiler to prevent many kinds of error - A constraint on how you write code - When you reference certain variables, they become "consumed" - A consumed variable cannot be referenced again later

Declaration of Variables
